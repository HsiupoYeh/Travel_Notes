# 真航空
+ 經濟艙種類只差在變更之類的手續費高低。
+ 台北->大邱:
  + 票價組成: 機票700up+燃油費968+桃園機場使用費500=2168up(最近最低約在2350元)
  + 飛機: 波音737
+ 大邱->台北:
  + 票價組成: 機票700up+燃油費583+大邱機場使用費543=1826up(最近最低約在1830元，時間不好應該要+500成本=2330元)
  + 飛機: 波音737
  + 回到台灣沒有交通工具
+ 期望票價:
  + 最便宜只可能出現3994元
  + 4000~5000就是很棒的促銷價
  + 5000~6000是期望票價
  + 以上都要另外+500的回程交通費
+ 其他:
  + KTX大邱->釜山:17,100韓幣(約400台幣)
